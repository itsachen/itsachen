EmoRecord

EmoRecord is a Java implementation of an emotional data logger for the Emotiv EPOC headset. It allows non-technical users to extract emotional data logged by the brain control interface through an intuitive GUI.

Stegafy

Stegafy is a free online steganography app. It was originally developed to test the graphics capabilities of PHP (GD).

notetag

notetag is a webapp that allows users to record lectures and meetings through a multi-platform mobile application, tag specific sections of these recordings, and upload them to a website where they can be subsequently shared with a community.

notetag was developed during a Facebook Camp Hackathon and went on to win first place at Cornell University. notetag is currently under development
